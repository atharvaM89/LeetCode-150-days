# Inheritance

## **1. Constructor, Getter, and Setter**

```java
class Account {
    private String name;
    private String phoneno;
    private String accountNo;
    private String address;
    private String DOB;
    protected double balance;

    // Constructor
    public Account(String n, String pno, String accno, String add, String dob) {
        name = n;
        phoneno = pno;
        accountNo = accno;
        address = add;
        DOB = dob;
        balance = 0.0;
    }

    // Getters
    public String getName() { return name; }
    public String getPhoneNo() { return phoneno; }
    public String getAccountNo() { return accountNo; }
    public String getAddress() { return address; }
    public String getDOB() { return DOB; }

    // Setters
    public void setName(String nm) { name = nm; }
    public void setPhoneNo(String phno) { phoneno = phno; }
    public void setAccountNo(String acno) { accountNo = acno; }
    public void setAddress(String add) { address = add; }
    public void setDOB(String db) { DOB = db; }
}

```

### **SavingAccount Class**

```java
class SavingAccount extends Account {
    public SavingAccount(String n, String pno, String accno, String add, String dob) {
        super(n, pno, accno, add, dob);
    }

    public void deposit(double amt) {
        balance += amt;
        System.out.println("Deposited: " + amt);
    }

    public void withdraw(double amt) {
        if (amt <= balance) {
            balance -= amt;
            System.out.println("Withdrawn: " + amt);
        } else {
            System.out.println("Insufficient balance!");
        }
    }

    public void showBalance() {
        System.out.println("Current Balance: " + balance);
    }
}

```

### **LoanAccount Class**

```java
class LoanAccount extends Account {
    public LoanAccount(String n, String pno, String accno, String add, String dob) {
        super(n, pno, accno, add, dob);
    }

    public void payEMI(double amt) {
        balance -= amt;
        System.out.println("EMI Paid: " + amt);
    }

    public void repay(double amt) {
        if (balance == amt) {
            balance = 0;
            System.out.println("Loan fully repaid!");
        } else {
            System.out.println("Remaining balance: " + balance);
        }
    }
}

```

## **2. Constructor in Inheritance**

```java
class Parent {
    public Parent() {
        System.out.println("Parent Constructor");
    }
}

class Child extends Parent {
    public Child() {
        System.out.println("Child Constructor");
    }
}

class GrandChild extends Child {
    public GrandChild() {
        System.out.println("GrandChild Constructor");
    }
}

public class InheritConst {
    public static void main(String[] args) {
        GrandChild c = new GrandChild();
    }
}

```

### **Output:**

```
Parent Constructor
Child Constructor
GrandChild Constructor

```

## **3.  Super in Parameterized Constructor**

```java
class Parent1 {
    public Parent1() {
        System.out.println("Non-parameterized Parent Constructor");
    }

    public Parent1(int x) {
        System.out.println("Parameterized Parent Constructor");
    }
}

class Child1 extends Parent1 {
    public Child1() {
        System.out.println("Non-parameterized Child Constructor");
    }

    public Child1(int y) {
        System.out.println("Parameterized Child Constructor");
    }

    public Child1(int x, int y) {
        super(x); // Calls Parent(int x)
        System.out.println("Parent + Child Parameterized Constructor");
    }
}

public class ParameterizedInheritance {
    public static void main(String[] args) {
        Child1 c = new Child1(5, 20);
    }
}

```

### **Output:**

```
Parameterized Parent Constructor
Parent + Child Parameterized Constructor

```

## **4. Concept of Super — Rectangle and Cuboid**

```java
class Rectangle {
    int length;
    int breadth;

    public Rectangle() {
        length = breadth = 1;
    }

    public Rectangle(int l, int b) {
        length = l;
        breadth = b;
    }
}

class Cuboid extends Rectangle {
    int height;

    public Cuboid() {
        height = 1;
    }

    public Cuboid(int h) {
        height = h;
    }

    public Cuboid(int l, int b, int h) {
        super(l, b);
        height = h;
    }

    int volume() {
        return length * breadth * height;
    }
}

public class CuboidInheritance {
    public static void main(String[] args) {
        Cuboid c = new Cuboid(5, 10, 15);
        System.out.println("Volume = " + c.volume());
    }
}

```

### **Output:**

```
Volume = 750

```

## **5.  Method Overriding**

```java
class TV {
    public void switchOn() {
        System.out.println("TV Switched On");
    }

    public void changeChannel() {
        System.out.println("TV Channel Changed");
    }
}

class SmartTV extends TV {
    @Override
    public void switchOn() {
        System.out.println("Smart TV Switched On");
    }

    @Override
    public void changeChannel() {
        System.out.println("Smart TV Channel Changed");
    }

    public void browse() {
        System.out.println("Smart TV Browsing Internet");
    }
}

public class Television {
    public static void main(String[] args) {
        SmartTV s = new SmartTV();
        s.switchOn();
        s.changeChannel();
        s.browse();
    }
}

```

### **Output:**

```
Smart TV Switched On
Smart TV Channel Changed
Smart TV Browsing Internet

```

## **6. Dynamic Method Dispatch**

```java
class TV2 {
    public void switchOn() {
        System.out.println("TV Switched On");
    }

    public void changeChannel() {
        System.out.println("TV Channel Changed");
    }
}

class SmartTV2 extends TV2 {
    @Override
    public void switchOn() {
        System.out.println("Smart TV Switched On");
    }

    @Override
    public void changeChannel() {
        System.out.println("Smart TV Channel Changed");
    }

    public void browse() {
        System.out.println("Smart TV Browsing Internet");
    }
}

public class DynamicDispatchDemo {
    public static void main(String[] args) {
        TV2 t = new SmartTV2(); // Reference of parent, object of child

        t.switchOn();       // SmartTV2 method is called
        t.changeChannel();  // SmartTV2 method is called

        // t.browse(); ❌ Not allowed (method not in parent class)
    }
}

```

### **Output:**

```
Smart TV Switched On
Smart TV Channel Changed

```

✅ **Explanation:**

- When a **parent reference** holds a **child object**,
    
    overridden methods of the **child** are called → **Dynamic Method Dispatch**.
    
- Extra methods (only in subclass) **cannot be called** via parent reference.
