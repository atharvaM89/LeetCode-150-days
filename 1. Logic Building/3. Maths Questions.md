## 4. Prime No

```java
import java.util.*;
public class App {
    public static void main(String[] args) throws Exception {
        System.out.println("Enter the No You want to check wheater it is prime or not");
        Scanner sc=new Scanner(System.in);
        
        int a=sc.nextInt();
        
        /* condition for non prime no -  Between 2 to  sqrt(n-1) There exisits an no which will completely divide it 
        if is it so then no is not a prime no
        otherwise it is a prime no  
        */
        
        
        
        for(int i=2; i<=Math.sqrt(a); i++){
            if(a%i==0){
                System.out.println("It is Not a Prime No");
            }
            else{
                System.out.println("It is a prime NO");
            }
        }
    }
}

```

## 7. Amstrong No

```
import java.util.*;
public class App {
    public static void main(String[] args) throws Exception {
        System.out.println("Enter the Number");
        Scanner sc=new Scanner(System.in);

        int no=sc.nextInt();
        
    
        int digit=0;
        int sumOfDigit=0;
        while(no!=0){
            digit=no%10;
            no=no/10;
            
            
            sumOfDigit+=(digit*digit*digit);
        }
        if(sumOfDigit==no){
            System.out.println("It is an Armstrong no");
        }
        else{
            System.out.println("It is Not and Armstrong no");
        }
    }
}

```
